>## __Armstrong numbers__


>_Pseudo code for armstrong numbers_ 

Start

* __DECLARE__ 

     *number* as integer

     *num* as integer

     *sum* as integer

     *digit* as integer

 * __OUTPUT__  ' What's the number '  

 * __INPUT__  user inputs the number

 * __ASSIGN__ 

      *num* = *number*

      *sum* = 0


        
 * __WHILE__    *number* > 0 ,

      *digit* = *number* % 10

      *sum* = *sum* + (*digit*)^3

      *number* = *number*/10

  * __UNTIL__    *number* <= 0  
  
    * __END WHILE__  

 * __IF__   *sum* == *num* , THEN 


 *   __OUTPUT__        ' a Armstrong number '

*  __ELSE__


   __OUTPUT__        ' not a Armstrong number '

  End
